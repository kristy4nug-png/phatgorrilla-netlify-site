# Brand Assets

These images are the current Phat Gorrilla visual direction for WordPress/WooCommerce.

## Images

- `built-different.png` - primary brand hero.
- `new-drop.png` - main shop/drop promotional graphic.
- `join-the-movement.png` - social and brand-follow graphic.
- `elite-range-black-gold.png` - Super Elite black and gold range graphic.
- `elite-colourways-neon.png` - Elite Colourways neon version.
- `elite-colourways-earth.png` - Elite Colourways earth-tone version.

## Colour Direction

Use a black base with high-contrast accents:

- `#050505` black
- `#141414` charcoal
- `#F91A7D` hot magenta
- `#BA0752` deep magenta
- `#A6C303` acid lime
- `#576D02` army green
- `#C99E41` metallic gold
- `#F7D87B` bright gold
- `#E3D4C0` cream
- `#C4A684` sand
- `#3D4E02` olive

## WordPress Usage

Upload these into WordPress Media Library and apply them to the homepage, shop page, Super Elite category, and launch/social sections. Do not use this folder as a website deployment source.
